Breathe2Sleep Spreadsheet Generator

What is Breathe2Sleep?

  • Patient portal for a self-management program for individuals with sleep apnea who are beginning Treatment with CPAP
  • Patients log their CPAP usage information
  • Used to tell how people use their CPAP machines and how effective it is
  • Provides a hub for CPAP information

Goals:


  • PHP script that pulls user data from the Breathe2Sleep database, normalizes/santizes the data, create spreadsheet using that information
  • Spreadsheets contain user-entered data and analytics data for any patient who took part in the study
  • Create simple website to interface with the PHP script to allow site administrators to download user spreadsheets
